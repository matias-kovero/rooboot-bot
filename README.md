### Installation

Rooboot-bot requires [Node.js](https://nodejs.org/) v8+ to run.

Install the dependencies and devDependencies and start the server.

```sh
$ npm install
$ npm start
```


## Supported commands

  - /piato /lozzi /maija /libri /tilia /syke /rentukka /ylisto /fiilu /ilokivi
  - /laulu
